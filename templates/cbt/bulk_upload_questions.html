Question Structure

  1. Start each question with Q1., Q2., etc.
  2. Use A. B. C. D. for options
  3. Mark the answer: ANSWER: A
  4. Optional marks: MARKS: 2
  5. For passages: PASSAGE TITLE:, START PASSAGE, END PASSAGE
  6. For passage groups: PASSAGE GROUP: group_id
  7. For images: IMAGE: filename.jpg and CAPTION: description
  8. Word subscript (X₂), superscript (X²), bold, italic work natively

Example with Passage

PASSAGE TITLE: The Water Cycle
PASSAGE GROUP: passage_1
START PASSAGE
Water evaporates from oceans, lakes, and rivers.
It rises into the atmosphere as water vapor.
When it cools, it condenses into clouds.
END PASSAGE

Q1. What happens to water when it evaporates?
A. It becomes ice
B. It rises as water vapor
C. It falls as rain
D. It becomes a cloud
ANSWER: B
MARKS: 2

Q2. What process forms clouds?
IMAGE: cloud_formation.jpg
CAPTION: Cloud Formation Process
A. Evaporation
B. Condensation
C. Precipitation
D. Collection
ANSWER: B

Required Columns

  • question_text — The question text
  • question_type — MCQ, TF, or SA
  • marks — Point value
  • option_a, option_b, option_c, option_d, option_e — Answer options (e is optional)
  • correct_option — Correct answer letter (A, B, C, or D)
  • correct_answer — For Short Answer questions only

How It Works

  • Each question must start with a number: 1. 1) Q1. (1)
  • Options use any of: A. A) A: (A)
  • Answer line: ANSWER: B or ANS: B or The answer is B
  • Marks line (optional): MARK: 0.5 or MARKS: 1 or POINTS: 2
  • Separate questions with a blank line, or just start the next number
